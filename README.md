# Toronto Superior Detailing & Rustproofing — 1-Page Site

This is a fast, mobile-first, conversion-focused one-page website where the **primary goal is phone calls**.

## Run locally

- Open `index.html` directly in a browser, or
- Use any simple static server (recommended):

```bash
npx serve .
```

## Add your provided images (required)

Drop your real business photos into `assets/` using these filenames (the page already references them):

- `assets/hero.webp` (hero background image)
- `assets/why.webp` (photo in “Why Choose Us” section)

Tips:
- Use **WebP** for speed.
- Aim for ~200–400KB each if possible.
- Recommended sizes:
  - `hero.webp`: ~1600px wide
  - `why.webp`: ~1200px wide
